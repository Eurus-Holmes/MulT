# NLP 概述
### 解决 NLP 问题的一般思路

这个问题人类可以做好么？

  - 可以 -> 记录自己的思路 -> 设计流程让机器完成你的思路
  
  - 很难 -> 尝试从计算机的角度来思考问题


### NLP 的历史进程
- **规则系统**
    - 正则表达式/自动机
    - 规则是固定的
    - **搜索引擎**
        ```tex
        “豆瓣酱用英语怎么说？”
        规则：“xx用英语怎么说？” => translate(XX, English)
        
        “我饿了”
        规则：“我饿（死）了” => recommend(饭店，地点)
        ```
- **概率系统**
    - 规则从数据中**抽取**
    - 规则是有**概率**的
    - 概率系统的一般**工作方式**
        ```tex
        流程设计
        收集训练数据
            预处理
            特征工程
                分类器（机器学习算法）
                预测
                    评价
        ```
